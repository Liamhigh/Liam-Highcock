# Clone and Build - Simplest Instructions

Want to build this Android app? Here's how in 3 steps:

## Prerequisites
- Install [Android Studio](https://developer.android.com/studio) (includes everything you need)

## Step 1: Clone This Branch
```bash
git clone -b copilot/create-complete-build-branch https://github.com/Liamhigh/Liam-Highcock.git
```

## Step 2: Open in Android Studio
1. Launch Android Studio
2. Click "Open"
3. Select the `Liam-Highcock` folder you just cloned
4. Wait for sync (5-10 minutes first time)

## Step 3: Build
Click: **Build > Build Bundle(s) / APK(s) > Build APK(s)**

Done! Your APK is at: `app/build/outputs/apk/debug/app-debug.apk`
